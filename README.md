## State Management Approach
I used the GetX state management solution because of it's fast and easy implementation which seems best for this project.
With this State management solution, I could easily seperate my business logic from the UI and making it reactive and easy to maintain.

## Setup Instructions
1. Clone repository
2. Run `flutter pub get`
3. Run `flutter run`
